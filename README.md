# MacOS install (Brewfile & Dotfiles)

## Getting Started

### Prerequisites

- Install macOS
- Complete initial guide

### Installation

- Clone repository to ~ `git clone https://github.com/mirkonz/.dotfiles.git` (Repo needs to be public)
- `cd .dotfiles`
- Run `sh ./install.sh`
